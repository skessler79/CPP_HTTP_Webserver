Welcome to My Simple HTML Demo!

This is a demonstration of some basic HTML features:

  • Headings: Using h1, h2, h3, etc. for different levels of headings.
  • Paragraphs: Using the p tag for paragraphs of text.
  • Lists: Ordered and unordered lists using ul and ol tags.
  • Links: Creating links using the a tag.
  • Images: Embedding images using the img tag.
  • Styling: Adding styles with inline CSS or external CSS files.

Feel free to explore and modify this HTML code!
